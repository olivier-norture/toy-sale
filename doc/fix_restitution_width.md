# Make tables in restitution_jouet.php responsive

This change makes the tables in `web/restitution_jouet.php` responsive.

The changes include:
- Setting the table width to 100%.
- Removing fixed widths from table header cells in `main.css`.
- Using percentage-based widths for columns in `style.css` to allow for better responsiveness.
